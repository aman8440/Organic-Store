I have use full stack technologoies.
## Technologies:-
1. HTML
2. CSS
3. JavaScript
4. Nodejs
5. Expressjs
6. MongoDB
7. Bootstrap
8. Ejs
9. Tailwind CSS

## How to start?
1. You will install Nodejs and mongoDB
2. Install all the dependencies using npm
3. Run all the command using nodemon
